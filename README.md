<h2>Installation</h2>

1. Clone the source code:

```
git clone https://github.com/KennethEvolves/s-chat-bot-app.git
```

2. Go to project:

```
cd s-chat-bot-app
```

3. Create a new branch:

```
git checkout -b name_new_branch
```

4. Update new branch:

```
git pull origin tag/chatbot-beta
```

5. Install development dependencies:

```
yarn install
```

6. Run application:

```
yarn dev
```
